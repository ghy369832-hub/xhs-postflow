# Evidence Chain

Use this file after `yyl-benchmark-breakdown` has produced or attempted to produce a report.

## Required Evidence Pack

```markdown
## 证据包
- 来源链接:
- 平台 / 类型:
- 作者 / 标题:
- 数据: 赞 / 藏 / 评 / 分享 / 发布时间; missing values as 未获取
- 取数方式:
- 口播/正文状态: 完整转写 / 原文 / 缺失
- 画面证据状态: 已读抽帧 / 已读图文 / 缺失

## 可复用规则
- 选题公式:
- 结构公式:
- 标题/封面公式:
- 视觉公式:
- 情绪/痛点:

## 改写边界
- 保留:
- 改写:
- 禁止照搬:
- 风险点:
```

## Evidence Standards

- Metrics must come from captured metadata; otherwise write `未获取`.
- Quotes must come from transcript/body; otherwise paraphrase and mark as inference.
- Visual claims require inspected frames/images.
- Do not use the competitor's unique personal story as the user's story unless the user explicitly confirms it.
- The transformation must create a new angle, examples, and wording while preserving the reusable structure.
